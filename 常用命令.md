## 如果提示连不上 github.com 443

先检查是不是网络问题：

```powershell
cd "d:\Trae工程\0408 个人网站 GAIA"
git ls-remote origin -h refs/heads/main
```

如果这一步失败，说明当前网络连不上 GitHub。

处理顺序：

1. 换网络
2. 开热点
3. 开代理或 VPN
4. 然后重新执行

```powershell
cd "d:\Trae工程\0408 个人网站 GAIA"
git push origin main
```
