## 注意事项

- Vite 会将 `public/` 目录下的文件原样复制到构建输出
- `src` 路径以 `/audio/` 开头，对应 `public/audio/` 目录
- 支持 MP3、OGG、WAV 格式（推荐 MP3 兼容性最好）
